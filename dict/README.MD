
Словник — це колекція, яка зберігає дані у форматі «ключ: значення».
Уяви звичайну записну книжку: ім'я друга — це ключ, а його номер телефону — значення.

Ключі завжди мають бути унікальними.

Доступ до даних відбувається за ключем (як за індексом у списках, але замість цифр — назви).

🛠 Основні методи на прикладах
Створимо словник для нашого учня:

Python
student = {
    "name": "Іван",
    "age": 16,
    "course": "Python"
}
1. Метод .get() — безпечний доступ
Дозволяє взяти значення за ключем. Якщо такого ключа немає, програма не «впаде» з помилкою, а поверне None або ваш текст.

Python
# Шукаємо оцінку, якої немає в словнику
grade = student.get("grade", "Оцінка ще не виставлена")
print(grade)  # Результат: Оцінка ще не виставлена
2. Метод .keys() — тільки заголовки
Повертає список усіх ключів 

Python
print(student.keys()) 
# Результат: dict_keys(['name', 'age', 'course'])

3. Метод .values() — тільки дані
Повертає список усіх значень, які зберігаються у словнику.

Python
print(student.values()) 
# Результат: dict_values(['Іван', 16, 'Python']) 
dict_values([]) це це динамічне "вікно" у твій словник. тобто не сам список, Навіщо це? Щоб не витрачати пам'ять комп'ютера на створення окремої копії списку ключів. 
Як з нього створити реальнйи список ? Треба використати метод list(student.values()) 

Якщо дуже просто 
Уяви, що твій словник — це кімната, а ключі — це речі в ній.

Список list — це як фотографія цих речей. Якщо ти винесеш шафу з кімнати, на фото вона все одно залишиться. Це окрема копія.

Подання dict_keys — це як дзеркало, що висить навпроти речей. Якщо ти прибереш шафу, вона миттєво зникне і в дзеркалі.

Чому Python так робить?
Комп'ютер каже: "Навіщо мені витрачати папір і час на друк фотографії (списку), якщо я можу просто дати тобі дзеркало і ти побачиш усе, що є в кімнаті зараз?".

Головне запам'ятати:
Дзеркало (dict_keys) — легке і швидке, але ти не можеш тицьнути пальцем у "перший предмет у дзеркалі" (немає індексів [0]).

Фотографія (list) — займає пам'ять, але з нею можна робити що завгодно окремо від кімнати.

4. Метод .items() — все разом
Повертає пари (кортежі), де є і ключ, і значення. Ідеально для циклів.

Python
for key, value in student.items():
    print(f"Поле {key} має значення {value}")

def get_price(item):
    return item["price"]

products = [
    {"name": "Огірок", "price": 45},
    {"name": "Помідор", "price": 80},
    {"name": "Капуста", "price": 25}
]

# Шукаємо МІНІМУМ, використовуючи ціну як критерій
cheapest_item = min(products, key=get_price)

print(f"Найдешевший овоч: {cheapest_item['name']} (ціна: {cheapest_item['price']})")
