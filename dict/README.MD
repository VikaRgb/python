Задача 1: "Менеджер покупок" (Списки)

Умова: У вас є список продуктів. Напишіть програму, яка додає новий продукт у кінець, видаляє продукт за назвою та виводить список у відсортованому алфавітному порядку.

Вхідні дані: items = ["Яблуко", "Молоко", "Хліб"]

Дія: Додати "Сир", видалити "Молоко".

Очікуваний результат: ['Сир', 'Хліб', 'Яблуко']

Підказка: Використовуйте методи .append() для додавання, .remove() для видалення та функцію sorted() або метод .sort() для впорядкування.

Задача 2: "Електронний журнал" (Словники)

Умова: Створіть словник, де ключі — імена студентів, а значення — їхні оцінки. Напишіть код, який знаходить середній бал усіх студентів.

Вхідні дані: journal = {"Олег": 90, "Марія": 95, "Іван": 85}

Очікуваний результат: Середній бал: 90.0

Підказка: Ви можете отримати всі оцінки за допомогою journal.values(). Для розрахунку використайте sum() (сума всіх елементів) та len() (кількість елементів).

Задача 3: "Координати об'єктів" (Кортежі)

Умова: Дано список кортежів, де кожен кортеж — це координати (x, y). Напишіть програму, яка виведе лише ті точки, які знаходяться в першій чверті графіка (де і x, і y більше 0).

Вхідні дані: points = [(1, 5), (-2, 3), (4, -1), (10, 2)]

Очікуваний результат: (1, 5), (10, 2)

Підказка: Використовуйте цикл for point in points:. Доступ до елементів кортежу здійснюється за індексами: point[0] для x та point[1] для y.

Задача 4: "Конвертер даних" (Комбінована)

Умова: Маємо список кортежів (id, name). Перетворіть його на словник, де id буде ключем, а name — значенням.

Вхідні дані: users_list = [(101, "Admin"), (102, "User1"), (103, "Moderator")]

Очікуваний результат: {101: 'Admin', 102: 'User1', 103: 'Moderator'}

Підказка: Спробуйте використати функцію dict(), яка вміє автоматично перетворювати список парних елементів у словник.

Задача 5: "Унікальні слова" (Списки та Словники)

Умова: Дано рядок тексту. Порахуйте, скільки разів кожне слово зустрічається в тексті, і збережіть результат у словник.

Вхідні дані: "apple banana apple cherry banana apple"

Очікуваний результат: {'apple': 3, 'banana': 2, 'cherry': 1}

Підказка: Спочатку перетворіть рядок на список слів за допомогою .split(). Потім пройдіть циклом по списку і додавайте слова в словник, збільшуючи лічильник.
