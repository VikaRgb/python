Это параметры, уже имеющие готовые значения. Если мы забудем или не захотим их указывать при вызове функции, она сама возьмет значение "по умолчанию".
def make_coffee(type="Эспрессо", milk="коровье"): 
print(f"Приготовлено {type} на {milk} молоке.")

# 1. Вызываем без аргументов – берутся значения по умолчанию
make_coffee() # Выведет: Приготовлено Эспрессо на коровьем молоке.

#2. Меняем только тип кофе
make_coffee("Капучино") # Выведет: Приготовлен Капучино на коровьем молоке.

#3. Меняем все полностью
make_coffee("Латте", "кокосовом") # Выведет: Приготовлен Латте на кокосовом молоке.

Именуемые аргументы (Keyword Arguments)
Обычно Python передает аргументы по порядку (позиционные аргументы). Но мы можем четко выделить каждый аргумент во время вызова. Тогда их порядок становится совершенно неважным!

def create_character(name, role, level): 
print(f"Создан героя {name}. Класс: {role}, Уровень: {level}")

# Передаем аргументы, путая порядок, но указывая имена:
create_character(level=80, role="Маг", name="Гендальф")
# Все работает идеально!


Переменное количество аргументов (*args и kwargs)
Иногда мы не знаем, сколько данных передаст пользователь. Например, сколько топпингов он выберет для пиццы или сколько вещей положит в рюкзак.

📦 *args (Передаем много обычных значений)
Звездочка перед словом args (имя может быть любым, например *items, но args — это стандарт) говорит Python: "Собери все дополнительные аргументы в один кортеж (tuple)".

def make_pizza(size, *toppings): 
print(f"Делаем пиццу размером {size} см со следующими ингредиентами:") 
for topping in toppings: 
print(f"-{topping}")

# Можем передать 2 ингредиента:
make_pizza(30, "сыр", "пепперони")

# А можем 4:
make_pizza(45, "грибы", "ветчина", "кукуруза", "оливки")

kwargs (передаем много именуемых значений)
Две звездочки собирают все именуемые аргументы (параметры типа ключ = значение) в один словарь (dict).

def show_profile(name, **additional_info): 
print(f"Пользователь: {name}") 
#custom_info теперь является словарем 
for key, value in additional_info.items(): 
print(f"{key}: {value}")

# Передаем столько дополнительных данных, сколько хотим
show_profile("Елена", age=16, city="Киев", hobby="рисование")
