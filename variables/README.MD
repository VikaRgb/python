Конспект: Змінні в Python 🐍

Змінна — це "коробка" з назвою, у якій зберігається певна інформація (дані).

1. Як створити змінну?

У Python змінні створюються за допомогою знака дорівнює =. Це називається оператором присвоювання. 
name = "Олексій"  # Змінна name зберігає текст
age = 14           # Змінна age зберігає число 

2. Правила іменування (Як давати назви)

Щоб програма працювала правильно, назви змінних повинні відповідати правилам:

Можна: використовувати латинські літери (a-z), цифри та нижнє підкреслення _.

Не можна: починати назву з цифри (наприклад, 1step — помилка).

Не можна: використовувати пробіли (замість них — user_name).

Регістр важливий: Age та age — це дві різні змінні.

3. Основні типи даних

У Python змінна сама розуміє, який тип даних вона зберігає:

int (integer) — цілі числа: 5, -10, 1000.

float — дробові числа (записуються через крапку): 3.14, 0.5.

str (string) — рядки (текст), завжди в лапках: "Привіт", 'Python'.

bool (boolean) — логічний тип, має лише два значення: True (істина) або False (хибність).

4. Динамічна типізація

Ви можете змінити тип змінної в будь-який момент, просто записавши туди нове значення:

x = 5       # зараз x — це число
x = "Київ"  # тепер x — це рядок


5. Корисні команди

print(назва_змінної) — вивести значення на екран.

type(назва_змінної) — дізнатися, який тип даних у змінної.

input() — дозволяє користувачеві самому ввести дані з клавіатури.

Приклад:

user_name = input("Як тебе звати? ")
print("Привіт, " + user_name + "!")
